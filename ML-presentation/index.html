Machine Learning(overview)

What is Machine Learning?

“Machine learning (ML) is a field of inquiry devoted to understanding and building methods that 'learn', that is, methods that leverage data to improve performance on some set of tasks” - Wiki

The Brief History of Machine Learning

1949 - "The Organization of Behavior" book by Donald Hebb - an entry point for Machine Learning studies.
1950 - Alan turing's paper "Computing Machinery And Intelligence" - first introduction of the Turing Test(The Imitation Game).
1952 - Arthur Samuel developed the first self-learning computer program, for playing checkers.
1957 - Frank Rosenblatt designed “The Perceptron” - a computer model, which simulated the thought process of the human brain.
1997 - IBM's computer Deep Blue beats the world champion Harry Kasparov in the game of chess.

Later, humans also got beaten up in Jeopardy, Go, all Atari games, Poker, StarCraft and Dota2.

2014 - Facebook developed DeepFace - a software algorithm, capable of recognizing and identifying human's faces in digital images with 97 percent accuracy.
2015 - Amazon launches Amazon Machine Learning, Google launches Distributed Learning Machine Toolkit.
Present days - Machine Learning is EVERYWHERE - in software development, business analysis, advertising, education, healthcare.
